# Agent-Built Agent

A fully self-editing AI agent that can rewrite its codebase and restart itself.

## Features

- Self-modifying code
- Automatic restart capability
- Edit tracking and logging

## Usage

```bash
python main.py
```

## Architecture

The agent consists of:
- `SelfEditingAgent` class: Core agent logic
- Codebase modification utilities
- Restart and status tracking
